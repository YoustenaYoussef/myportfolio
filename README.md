# 🌐 My Portfolio Website

## 📌 About the Project
This is a simple personal portfolio website built using HTML and CSS.  
It showcases my skills, projects, and contact information in a clean and responsive design.

## ✨ Features
- Responsive design for all devices
- About section to introduce myself
- Skills section
- Projects showcase
- Contact section

## 🛠️ Built With
- HTML5
- CSS3

## 🚀 How to Run
Simply open the `index.html` file in any web browser.
